# Theia

[![Build status](https://travis-ci.org/carldata/theia.svg?branch=master)](https://travis-ci.org/carldata/theia)

Data generator for Hydra system.

 
## Running test
 
 ```bash
sbt assembly
java -jar target/scala-2.12/theia.jar 
 ```
